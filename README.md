Gestor de Tareas en C++
Este proyecto es un programa sencillo y práctico escrito en C++ que permite gestionar una lista de tareas desde la consola. El usuario puede agregar nuevas tareas, ver todas las tareas registradas, marcarlas como completadas o eliminarlas.
Es un proyecto ideal para principiantes que quieren practicar programación estructurada, clases, vectores y menús interactivos.

Características principales
Añadir nuevas tareas a la lista

Mostrar todas las tareas registradas

Marcar tareas como completadas

Eliminar tareas por número

Menú interactivo fácil de usar

Código claro y entendible

Funcionamiento del programa
El usuario selecciona una opción del menú.

Según la opción elegida, puede:

Escribir una nueva tarea

Ver todas las tareas

Completar una tarea existente

Eliminar una tarea

El programa continúa ejecutándose hasta que el usuario elige salir.
